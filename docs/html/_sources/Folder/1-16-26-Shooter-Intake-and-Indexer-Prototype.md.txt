## Procedure for Creating a Shooter, Intake, and Indexer Subsystem
1. Create a new branch named Prototype
2. Create four new classes under subsystems
3. Name the classes CounterRoller, IndexerPrototype, IntakePrototype, ShooterPrototype
4. Add the frc package and imports
5. Each class extends TemplateSubsystem
6. We added flywheel constants and the configurations for them
7. We added logging
8. We added our new configured classes to RobotContainer
9. added the new classes in subsystems
10. added the parrallel commands under driver binding using run and runOnce
